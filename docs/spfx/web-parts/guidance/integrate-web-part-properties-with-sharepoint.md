<span data-ttu-id="d3726-p106">In klassischen SharePoint-Webparts waren Werte von Webpart-Eigenschaftswerten von SharePoint isoliert. Wenn Sie eine Webparteigenschaft mit einer URL einer Datei in SharePoint gespeichert hatten, mussten Sie manuell sicherstellen, dass diese URL gültig war und auf ein korrektes Dokument verwies, falls es verschoben oder umbenannt wurde. <span data-ttu-id="d3726-122">Wenn Sie Webparts erstellen, können Sie mit SharePoint-Framework bestimmen, welche Art von Wert die jeweilige Webparteigenschaft enthalten soll.</span><span class="sxs-lookup"><span data-stu-id="d3726-122">When building web parts, SharePoint Framework allows you to specify what kind of value the particular web part property holds.</span></span> <span data-ttu-id="d3726-123">Diese Konfiguration bestimmt, wie SharePoint den Wert verarbeitet.</span><span class="sxs-lookup"><span data-stu-id="d3726-123">This configuration determines how SharePoint handles the value.</span></span> <span data-ttu-id="d3726-124">Abhängig von der angegebenen Konfiguration kann SharePoint den Wert der jeweiligen Eigenschaft im Suchindex enthalten, unsichere HTML-Inhalte entfernen und sogar Links zu Dokumenten in SharePoint auf dem neuesten Stand speichern, falls eine Datei verschoben oder umbenannt wird.</span><span class="sxs-lookup"><span data-stu-id="d3726-124">When building web parts, SharePoint Framework allows you to specify what kind of value the particular web part property holds. <span data-ttu-id="d3726-125">Um die Konfiguration für Ihre Webparteigenschaften anzugeben, setzen Sie in der Webpartklasse den **propertiesMetadata**-Getter außer Kraft:</span><span class="sxs-lookup"><span data-stu-id="d3726-125">To specify the configuration for your web part properties, in the web part class, override the **propertiesMetadata** getter:</span></span>

```ts

  BaseClientSideWebPart,
  IPropertyPaneConfiguration,
  PropertyPaneTextField,
  IWebPartPropertiesMetadata
} from '@microsoft/sp-webpart-base';

// ...

export default class ArticleLinkWebPart extends BaseClientSideWebPart<IArticleLinkWebPartProps> {
  // ...
  protected get propertiesMetadata(): IWebPartPropertiesMetadata {
    return {
      'title': { isSearchablePlainText: true },
      'intro': { isHtmlString: true },
      'image': { isImageSource: true },
      'url': { isLink: true }
    };
  }
  // ...
}
```
