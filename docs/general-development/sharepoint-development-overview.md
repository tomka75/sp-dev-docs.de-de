<span data-ttu-id="adf54-112"><a name="bk_whatkinds"> </a></span><span class="sxs-lookup"><span data-stu-id="adf54-112"></span></span>

<span data-ttu-id="adf54-p103">Entwickler, die mit SharePoint vertraut sind, können bereits serverseitige Farm-Lösungen erstellen, mit denen die wichtigsten SharePoint-Funktionen erweitert werden. Mit dem neuen SharePoint-Add-In-Entwicklungsmodell können Sie Add-Ins erstellen, die SharePoint-Funktionen nutzen und in der Cloud statt auf Ihrer SharePoint-Farm ausgeführt werden. And SharePoint provides you with functionality to interact with SharePoint resources and a wide range of hosting options. The new SharePoint Add-in development model gives you the ability to build add-ins that take advantage of SharePoint capabilities and that run in the cloud instead of on your SharePoint farm. This flexible development model, along with the integration of standard web technologies, makes SharePoint development work more like other kinds of web development that you may already be doing.</span></span>
